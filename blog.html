Theme Modifications

I chose to work with the 2023 theme as it provided many options for me to customise my solution. It also integrates well with Elementor and Gutenberg page builders, and has a clean, user friendly interface.

1. Set up

I turned on Elementor’s Flexbox Container and Nested Elements feature so I could create a flexible, responsive layout. It helped me distribute space, align elements and nest elements inside each other.

I also enabled the ‘Post Name’ permalink structure so I would have clean and SEO friendly urls.

2. Further customisations

Further customisations I made included:

  • Adding custom styles to the global theme in Gutenberg
  • Using the Create Block Theme plugin to add the Google Fonts I wanted to work with to my solution
  • Using template parts to create the header and footer sections for my website
  • Using the Single post template to create a template for my individual blog posts. This allowed me to concentrate on the content when writing the posts rather than the layout
  • Adding additional CSS through the Custom CSS and JS plugin to allow for customised styling

JavaScript Additions

Places I enjoyed experimenting with code during the "deliver" part of the project included:

1. Image slider

1. I used JavaScript to create an automatic image slider on my homepage. I did this to grab the users’ attention and showcase what the site is about.

I did this by using a custom HMTL widget in Elementor, and adding my code there. I included HTML code to add the images, and a <script></script> tag to add the JS.

2. Animate on Scroll

I added the Animate on Scroll library on the Single Post template in Appearance > Customise section of the solution. This is the template that I used for all my posts, and meant I only had to add the library once in one place to apply to all of my posts.

I added a <link> tag in a Custom HTML block to access the CSS and a <script> tag in another Custom HMTL block to access the JS and initialise the AOS.

Once the library was added to my solution, I could add effects from the library to photos in my posts. I did this by converting the images in my posts to HTML blocks, and adding data.aos to the code. I used the “fade-up” effect as I thought this worked well when scrolling through a longer text such as a blog post.
